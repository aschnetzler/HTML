IST 336: Web-Based Applications in Information Systems

Course Description

This course is designed to provide students with skills on the fundamentals of client-side web developement languages used to build professional websites, such as Hypertext Markup Language (HTML/HTML5), Cascading Style Sheets (CSS) and JavaScript. The course introduces Web standards, Web Design principles, and Web Design and Developement tools. Hands-on assignments will provide students with practical experience developing interactive Web pages and websites using client-side technologies.

Restricted to IST major. Prerequisite IST 209- Introduction to Programming
